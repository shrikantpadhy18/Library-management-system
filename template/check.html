This project is created By Ram Moorthy,Shrikant padhy and Vicky mishra.

The intention behind this project was to eliminate the card system and enable the process with efficient,reliable and record maintenance by using centralized database,thereby reducing the probability of data loss by introducing online library system.

This System tends to remove the concept of waiting in queue as well as filling the card in order to issue books.
